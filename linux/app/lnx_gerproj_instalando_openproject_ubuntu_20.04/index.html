    Instalando o OpenProject no Ububtu Linux 20.04¶

    Instalando o pacote do OpenProject¶

    Devemos iniciar o processo com a importação da chave PGP utilizada para assinar os pacotes: 

    cd ~
sudo wget -qO- https://dl.packager.io/srv/opf/openproject/key | sudo apt-key add -

    Adicionando o repositório do OpenProject a lista de repositórios utilizados pelo sistema

    sudo wget -O /etc/apt/sources.list.d/openproject.list \
  https://dl.packager.io/srv/opf/openproject/stable/10/installer/ubuntu/20.04.repo

    Atualizando a lista de pacotes disponíveis e instalando o pacote do OpenProject

    sudo apt-get update
sudo apt-get install openproject

    Configuração inicial¶

    Após instalar o pacote podemos seguir com a configuração inicial do OpenProject utilizando um wizard fornecido no pacote de instalação.

    sudo openproject configure

    OBS: O wizard salva as informações sobre as configurações escolhidas no arquivo /etc/openproject/installer.dat, assim, caso o wizard seja interrompido os valores salvos serão reutilizados e serão apresentados apenas os passos que ainda não tiverem sido executados.

    • Passo 1: Configuração do servidor Postgresql

    Para realizar a instalação local do PostgreSQL, devemos selecionar a opção Install e em seguida Ok

    • Passo 2: Configurando o Apache

    O OpenProject bem com um servidor ruby que atende as conexões locais. Para receber conexões externas precisamos configurar um servidor web que funcionará como um proxy, realizando o forward das conexões externas como o OpenProject application server

    Utilizando o wizard conseguimos instalar o Apache2 para realizar a função do proxy. Para isto devemos selecionar a opção Install e em seguida Ok.

    Será solicitado a confirmação do Fully qualified domain name do hosts, ajuste conforme necessário e selecione Ok.

    Para instalar o openproject em uma subpasta do nosso servidor web, devemos fornecer o Server path prefix, neste caso podemos utilizar /openproject

    Neste momento não habilitaremos o SSL, este processo será realizado posteriormente. Então devemos selecionar a opção no e em seguida OK.

    Também não realizaremos a configuração da integração com um servidor de SVN/Git. Assim podemos selecionar a opção skip e em seguida OK.

    • Passo 3: Configurando o envio de e-mail

    Para o processo de envio e-mail utilizaremos um servidor smtp que já está instalado na maquina. Estão podemos selecionar a opção sendmail e em seguida OK.

    Em seguida, devemos fornecer o endereço de e-mail de quem será o administrador da aplicação e em seguida OK.

    • Passo 4: Instalando o memcached

    Neste caso é importante realizar a instalação de um mecanismo de cache para auxiliar no funcionamento do ambiente. Assim devemos escolher a opção install e em seguida OK.

    Este foi o ultimo passo do questionário do Wizard, a partir de agora ele realizará a instalação e configuração do ambiente, conforme as opções que passamos para ale. O processo de instalação e configuração pode levar alguns minutos.

    No final do processo terá sido gerado o arquivo /openproject/installer.dat com as informações sobre o processo de instalação.
