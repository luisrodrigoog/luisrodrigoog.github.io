    Listas¶

    Listas Nao ordenadas¶

    Podemos utilizar - ou *

    Código:

    - George Washington
- John Adams
- Thomas Jefferson

    Resultado:

    • George Washington
    • John Adams
    • Thomas Jefferson

    Listas Ordenadas:¶

    Devemos colocar um numero na frente de cada linha

    Código:

    1. James Madison
2. James Monroe
3. John Quincy Adams

    Resultado:

    1. James Madison
    2. James Monroe
    3. John Quincy Adams

    Listas aninhadas¶

    Código:

    1. First list item
    - First nested list item
        - Second nested list item

    Resultado:

    1. First list item
      • First nested list item
        • Second nested list item

    Lista de Tarefas¶

    Código:

    - [x] Finish my changes
- [ ] Push my commits to GitHub
- [ ] Open a pull request

    Resultado:

    • Finish my changes
    • Push my commits to GitHub
    • Open a pull request

    Extensão Admonition¶

    Esta extensão adicionar a possibilidade de utilizamos blocos de texto estilizados, que podem ser utilizados para destacar determinadas partes do conteúdo da página.

    Para mais informações sobre esta extensão consulte o site: python-markdown.

    Ativando a extensão¶

    Para ativar esta extensão devemos adicionar na seção markdown_extensions do arquivo mkdocs.yml, a seguinte linha: 

    1
2
    markdown_extensions:
   - admonition

    Utilizando a extensão¶

    A sintaxe é bem simples. Todo bloco começa com !!!, seguindo de um espaço e do tipo do bloco. O conteúdo do bloco começa na linha a seguir e deve estar indentado por 4 espaços: 

    !!! note
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.

    Este código gera um bloco semelhante à:

    Note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor massa, nec semper lorem quam in massa.

    Para alterar o título do bloco devemos utilizar

    !!! note "Novo Título"

    Para remover o título podemos utilizar: 

    !!! note ""

    Podemos inserir um bloco dobrável (collapsible) trocando os sinais de exclamações (!)por sinais de interrogação (?). Por exeplo:

    ??? note
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.

    Este código gera um bloco semelhante à:

    Note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor massa, nec semper lorem quam in massa.

    Adicionando um sinal de + logo após aos sinais de ? o bloco aparecerá, por padrão, aberto. Por exemplo: 

    ???+ note
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.

    Este código gera um bloco semelhante à:

    Note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor massa, nec semper lorem quam in massa.

    A extensão Admonition suporta tipos de qualificadores definidos pelos usuários. Quando conbinado ao tema Material, a extensão suporta os seguintes tipos:

    • Notas : note, ou seealso (Comando)
    • Abstract : abstract, summary ou tldr
    • Informação : info ou todo (Configuração)
    • Dica : tip, hint, important
    • Sucesso : success, check ou done (Saida)
    • Pergunta : question, help ou faq
    • Aviso : warning, caution ou attention (Erro)
    • Falha : failure, fail ou missing
    • Perigo : danger ou error
    • Bug : bug
    • Exemplo : example
    • Citação : quote ou cite
