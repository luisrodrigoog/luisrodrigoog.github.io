    Ativando o WSL no Windows 10¶

    Para ativar o WSL inicie o PowerSell como administrador e execute o comando abaixo:

    Comando
    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

    Após executar o comando acima é necessário reiniciar o Windows

    Atualizando para a versão WSL2¶

    Antes de prossegui é importante verificar se estamos rodando a versão do windows necessário. Verifique se a versão instalada é a 2004 ou superior, build 19041.

    Podemos utilizar o comando abaixo para verificar a versão do nosso windows:

    Comando
    winver

    Caso estejamos rodando a versão correta, podemos habilitar o recurso "Plataforma de Máquina Virtual". Para isto precisamos iniciar o Powershell em modo administrador e executar o seguinte comando:

    Comando
    dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

    Este comando deve gerar uma saída semelhante à:

    Saida
    Ferramenta de Gerenciamento e Manutenção de Imagens de Implantação
Versão: 10.0.19041.329

Versão da Imagem: 10.0.19041.508

Habilitando recurso(s)
[==========================100.0%==========================]
A operação foi concluída com êxito.

    Será necessário reiniciar o windows para concluir o processo.

    Definindo o WSL 2 como a versão padrão¶

    Para definir a versão 2 do WSL como sendo a padrão precisamos abrir o PowerShell em modo administrador e executar o seguinte comando:

    Comando
    wsl --set-default-version 2

    Você poderá ver esta mensagem depois de executar esse comando: WSL 2 requires an update to its kernel component. For information please visit https://aka.ms/wsl2kernel. Siga o link (https://aka.ms/wsl2kernel) e instale o MSI dessa página em nossa documentação para instalar um kernel do Linux em seu computador para o WSL 2 usar. Depois de instalar o kernel, execute o comando novamente e ele deverá ser concluído com êxito sem mostrar a mensagem.
