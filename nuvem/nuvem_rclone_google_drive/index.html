    Conectando o Rclone ao Google Drive¶

    O rclone é uma aplicação que permite sincronizar arquivos hospedados localmente com contas em uma nuvem, como por exemplo no Google Drive. Este artigo explica como configurar o acesso a um Drive Compartilhado do Google.

    Durante o processo de configuração o rclone, caso seja utilizado será executado como um webserver de forma a coletar o token retornado pelo Google.

    Testando as configurações¶

    Para testar o acesso ao Drive podemos utilizar o comando abaixo. Ele apresentará a listagem dos diretórios contidos na pasta raiz do Drive:

    1
2
3
4
5
    rclone lsd Drive:
    -1 2020-06-13 20:37:57        -1 Apostilas
    -1 2020-06-13 19:36:04        -1 Cursos
    -1 2020-06-16 09:47:25        -1 Teste
    -1 2020-06-14 00:02:37        -1 Verificar

    Para listar os arquivos da pasta raiz podemos utilizar o comando abaixo. Este comando listará apenas os arquivos.

    1
        rclone ls Drive:
